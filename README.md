# ☕ Coffee Website

This is a responsive coffee shop website built using HTML, CSS, and JavaScript.

## 🔥 Features
- Home section
- About section
- Menu section
- Testimonials
- Gallery
- Contact form
- Smooth scrolling
- Responsive design (mobile + desktop)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Project Description
